# Spacious Text

A module and CLI app for making spaced out text

## Install

```bash
npm install spacious-text
```

## Usage

CLI:

```bash
space-text <Your Text> # < Y o u r  T e x t >
```

As a module:

```js
const spaceText = require('spacious-text');

spaceText('<Your Text>'); // < Y o u r  T e x t >
```
